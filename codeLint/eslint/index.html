# ESLint 概述

# 先决条件

Node.js(^12.22.0, ^14.17.0, or >=16.0.0)

# 安装与使用

使用 npm 或 yarn 安装 ESLint

npm install eslint --save-dev
#or
yarn add eslint --dev

您接下来应该创建配置文件，下面的命令可以方便的做到这一步

npm init @eslint/config
#or
yarn create @eslint/config
