# 使用

# 安装

# npm
npm install --save-dev --save-exact prettier
# yarn
yarn add --dev --exact prettier

# 配置文件

Prettier 支持多种配置文件格式(按优先级排序)

  • 在package.json文件中添加prettier关键字
  • .prettierrc(JSON 或 YAML 格式)
  • .prettierrc.json 或 .prettierrc.yml 或 .prettierrc.yaml 或 .prettierrc.json5
  • .prettierrc.js 或 .prettierrc.cjs 或 prettier.config.js 或 prettier.config.cjs(使用 module.exports 导出一个对象)
  • .prettierrc.toml

详细配置查看配置

# .prettierignore

使用.prettierignore可以完全忽略(即不重新格式化)某些文件和文件夹

# Ignore artifacts:
build
coverage

# Ignore all HTML files:
*.html

# CLI

# write

# 格式化全部文件
prettier --write .
# 格式化指定目录
prettier --write app/
# 格式化指定文件
prettier --write app/components/Button.js
# 格式化符合正则要求的文件
prettier --write "app/**/*.test.js"

该命令用于格式化当前目录及其子目录下所有 Prettier 支持的文件

# check

prettier --check .

--check只检查文件是否已经格式化，而不是覆盖它们。

# 与 Linters 结合

# eslint-config-prettier (opens new window)

  1. 安装
npm install --save-dev eslint-config-prettier
  1. 添加配置到.eslintrc.json中
{
  "extends": [
    // other configs ...
    "plugin:prettier/recommended"
  ]
}

# stylelint-config-prettier (opens new window)

  1. 安装
npm install --save-dev stylelint-config-prettier
  1. 添加配置到.stylelintrc.*文件中
{
  "extends": [
    // other configs ...
    "stylelint-config-prettier"
  ]
}
